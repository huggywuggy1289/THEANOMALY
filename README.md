## 🎬 소개

**《Escape the Classroom》**은 플레이어가 낯익은 공간에서 벌어지는 **미세한 이상현상**을 감지하여 생존을 도모하는,  
1인칭 시점의 **관찰 심리 스릴러**입니다.

> “틀린 그림 찾기”가 **생존의 조건**이 된다면?  
> 당신의 시선, 기억력, 그리고 판단력이 모두 시험받습니다.

## 🏠 게임 시스템

### 🔰 0레벨 (기억 단계)

- 이상현상이 없는 정상 상태의 공간을 자유 탐색
- 각 방의 배치, 소품, 조도, 오브젝트 위치 등을 **기억**

### 🕵️ 1레벨 (탐지 단계)

- 무작위로 1~5개의 이상현상 등장
- **카메라로 촬영**하여 이상 여부 판정
- 모든 이상현상을 찾아낸 후 **현관문을 통해 탈출**
- 이상현상 1개라도 놓치면 **0레벨로 리셋**

---

## 👻 엔티티 시스템

- 엔티티는 **이상 판정이 불가능한 존재**
- 외형은 인형/그림자/가구 등으로 위장
- **촬영 시 배터리 50% 즉시 소모**
- **일부 엔티티는 추적** → 후레쉬로 정지 가능 (배터리 소모 2배)
